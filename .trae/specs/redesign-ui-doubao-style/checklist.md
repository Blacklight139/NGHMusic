# Checklist

## 设计系统
- [ ] 四端均采用豆包风格（柔和浅色背景、圆角卡片 ≥12px、充足留白、柔和阴影、低饱和强调色）
- [ ] 桌面端深色侧栏已改为浅色柔和风格，主色不再是高饱和绿色
- [ ] 四端共享设计 token 并各自落地（配色/圆角/间距/字体层级一致）

## 音源导入（文件上传）
- [ ] 音源导入改为文件上传（文件选择器选 .json），移除纯文本框导入
- [ ] 上传合法音源文件后校验/适配通过并导入，出现在音源列表
- [ ] 上传非法文件提示具体错误原因且不导入

## 音源管理（LXMusic 风格）
- [ ] 设置页以列表展示所有音源，每项含名称、启用/禁用开关、来源标识、操作入口
- [ ] 开关启停立即生效并持久化
- [ ] 支持拖动或按钮调整优先级，聚合搜索按新顺序执行并持久化
- [ ] 删除音源带确认，删除后从列表移除且不再参与搜索/播放
- [ ] 导入后音源立即出现在列表

## 功能页卡片化
- [ ] 搜索/播放列表/收藏/排行榜/本地音乐/歌词/设置各页采用卡片化布局
- [ ] 播放组件栏改为柔和浮动卡片样式（圆角、阴影、与内容区间距）

## 后端微调
- [ ] SourceManager 暴露有序列表与优先级更新接口
- [ ] 桌面 Tauri command 提供音源列表/排序/删除/文件导入

## 交互
- [ ] 页面切换平滑稳定，无闪烁/卡顿
- [ ] 卡片 hover/press 有柔和反馈
